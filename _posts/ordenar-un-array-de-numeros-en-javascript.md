Como ordenar un array de números en Javascript :

```js
let arrayOfNumbers = [0,1,5,2,3,8,6,7,222,111,56,57,98];

console.log(arrayOfNumber.sort((a,b) => a-b))

// Output
[0, 1, 2, 3, 5, 6, 7, 8, 56, 57, 98, 111, 222]
```
