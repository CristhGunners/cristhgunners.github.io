Como obtener el numero menor y mayor en un Array de números en Javascript :

```js
const numbers = [6, 8, 1, 3, 9, 11, 2, 0];

console.log(Math.min(...numbers)); // 0
console.log(Math.max(...numbers)); // 11
```
